#🐦 X (Twitter) Remake
🧩 Project Summary
This project is a mini remake of Twitter (X) built entirely with HTML, CSS, and Vanilla JavaScript. It’s designed to demonstrate strong front-end skills in DOM manipulation, state management, and event-driven UI updates — all without using frameworks. It’s lightweight, responsive, and features a modern dark mode toggle.

✨ Features
* 📝 Post Tweets — Add new tweets instantly to the feed
* ❤️ Like Tweets — Toggle likes dynamically
* 🔁 Retweet Tweets — Increase or decrease retweet counts
* 💬 View Replies — Expand or collapse threaded replies
* 🗑️ Delete Tweets — Instantly remove tweets from the feed
* 🌙 Dark Mode — Switch themes seamlessly with a single toggle
* ⚡ Instant Rendering — Feed updates automatically with no reloads

🧱 Built With
Technology	Purpose
HTML5	Structure and layout
CSS3	Styling, layout, and dark mode
JavaScript (ES6)	Logic, rendering, and interactivity
UUID (via jspm.dev)	Generates unique tweet IDs
Font Awesome	Icons for UI actions
🧠 What I Learned
* Building a data-driven UI using plain JavaScript
* Handling event delegation efficiently
* Rendering dynamic HTML with template literals
* Managing UI state without frameworks
* Implementing a dark mode via class toggling
